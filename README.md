# Super Trunfo em C

## Como compilar
gcc batalha_cartas.c -o batalha_cartas

## Como executar
./super_trunfo

## Exemplo de uso

Digite os dados da Carta 1:
Estado (ex: SP): SP
Codigo: A01
Nome da cidade: Sao Paulo
Populacao: 12000000
Area (km2): 1521
PIB (em bilhoes): 800
Numero de pontos turisticos: 20

Digite os dados da Carta 2:
Estado (ex: RJ): RJ
Codigo: B01
Nome da cidade: Rio de Janeiro
Populacao: 6700000
Area (km2): 1182
PIB (em bilhoes): 400
Numero de pontos turisticos: 30
